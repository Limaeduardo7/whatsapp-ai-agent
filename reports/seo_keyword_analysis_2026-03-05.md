# SEO Keyword Analysis — 2026-03-05

## Objetivo do turno
Executar rodada tática de keywords para acelerar aplicação on-page em páginas comerciais do ecossistema Meraki.

## Metodologia rápida
- Base: clusters já consolidados em 2026-03-02 e 2026-03-04.
- Priorização por: intenção comercial, potencial de CTR e facilidade de implementação on-page imediata.

## Clusters priorizados (execução imediata)

### 1) Cluster "site para empresa" (fundo de funil)
- site para empresa
- criação de site profissional
- orçamento site empresarial
- desenvolvimento de site para negócios

**Intenção:** Comercial transacional
**Ação SEO:** usar em páginas de serviço e landing de orçamento.

### 2) Cluster "landing page" (fundo/meio)
- criação de landing page
- landing page para captação de leads
- landing page para tráfego pago
- página de vendas profissional

**Intenção:** Comercial + solução específica
**Ação SEO:** aplicar em páginas de oferta orientadas a performance.

### 3) Cluster "site + IA" (categoria emergente)
- site com inteligência artificial
- automação de atendimento no site

- ia para pequenas empresas atendimento

**Intenção:** Comercial consultiva
**Ação SEO:** criar/otimizar seção dedicada com proposta de valor híbrida (site + automação).

## Priorização de páginas (ordem de ataque)
1. Página de serviço principal (website institucional/comercial)
2. Página de landing pages
3. Página/ seção de solução "Website + IA"

## Regras de otimização semântica
- 1 keyword primária por URL + 3-5 secundárias semanticamente próximas.
- Evitar canibalização: separar claramente intenção "site institucional" vs "landing page".
- Inserir termos de confiança comercial: prazo, portfólio, conversão, suporte.
