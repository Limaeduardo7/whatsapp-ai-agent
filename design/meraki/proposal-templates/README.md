# Design: Templates visuais para propostas comerciais

Entregáveis deste turno:
- `proposal-template-tokens.css` — tokens de cor/tipografia/componentes para proposta premium.
- `proposta_template_premium.md` — estrutura visual/copy-ready de proposta comercial.

Uso rápido:
1. Aplicar `proposal-template-tokens.css` na camada visual (HTML/PDF builder/Canva guideline).
2. Preencher placeholders do `.md` com dados do cliente.
3. Exportar em PDF ou imagem mantendo hierarquia visual e CTA final.
